# Ballad
A simple but beautiful text editor made using JavaFX.

# Features
- Themes support using CSS
- Flat design
- Line numbers
- Cut, Copy, Paste
- Autosave files
- File viewer

# Goals
- Syntax highlighting
- Auto-indentation
- More themes
- Find, Replace, Replace All

# Dependencies
- Needs Java 8 because of use of lambda expressions
- RichTextFX (https://github.com/TomasMikula/RichTextFX)
